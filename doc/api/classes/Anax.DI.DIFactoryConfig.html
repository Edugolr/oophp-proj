\Anax\DI

\Anax\DIDIFactoryConfig

DI factory class creating a set of default services by loading them from a configuration array, file and/or directory.

The general configuration array for a service looks like this. [ "services" => [ "request" => [ "active" => false, "shared" => false, "callback" => function () { $object = new \stdClass(); return $object; } ], ], ]
